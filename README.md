# Student Attendance Management System

A full-stack Student Attendance Management System built with React (Vite), Node.js, Express, PostgreSQL, and Sequelize.

## Tech Stack
- Frontend: React, Vite, Tailwind CSS, React Router, Axios, Recharts
- Backend: Node.js, Express, Sequelize, PostgreSQL, JWT
- Auth: JWT with Role-Based Access Control (Admin, Teacher, Student)
